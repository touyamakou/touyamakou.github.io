## Intro {#intro}
Jekyll and GitHub Pages make it easy to publish a small personal blog by adding Markdown files to the repository.

## Writing posts {#write}
Add a new post under the `_posts` directory.

```text
YYYY-MM-DD-title.md
```

You can set the title, description, topics, social preview image, and emoji in front matter.

```yaml
title: "Post title"
description: "Short description"
tags: [dev, memo]
topics:
  - name: dev
    icon: dev
image: "/assets/images/post-image.png"
emoji: "idea"
```

## Design {#design}
The profile page uses a Zenn-like header, tabs, and article cards. The article page follows the Zenn post layout with a centered article, a left action rail, and a right author and contents rail.
