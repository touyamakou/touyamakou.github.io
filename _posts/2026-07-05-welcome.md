Jekyll と GitHub Pages を使うと、Markdown ファイルを追加するだけでブログを更新できます。

## 記事を書く

新しい記事は `_posts` フォルダに追加します。

```text
YYYY-MM-DD-title.md
```

front matter にタイトル、説明、タグ、絵文字を設定できます。

```yaml
title: "記事タイトル"
description: "短い説明"
tags: [dev, memo]
emoji: "💡"
```

## デザイン

プロフィール、タブ、カード型の記事一覧を Zenn の個人ページに近い構成でまとめています。
