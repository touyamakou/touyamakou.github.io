# Zenn Style Jekyll Blog

GitHub Pages compatible Jekyll blog with a Zenn-inspired personal page and article page design.

## Local preview

```powershell
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.

## Customize

- Edit `_config.yml` for site/profile metadata.
- Add posts under `_posts/YYYY-MM-DD-title.md`.
- Replace `assets/images/avatar.svg` with your own avatar if desired.
